# Программа для анализа ТУ
_Инновент_

Для запуска необходимо:
- rust
- gtk4 всё библиотеки (окей гугл, всё найдётся)
- запущенная ollama с установленной моделью `gpt-oss:20b`

Запуск - `cargo run`

Работает на:
* Windows
* Linux

Чтобы распространять необходимо с исполняемым файлом поместить папку assets, а также зависимости gtk (можно посмотреть в моём релизном архиве)

_В дальнейшем планирую переделать это в установщик_
